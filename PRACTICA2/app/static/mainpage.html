Criba de Eratóstenes

La criba de Eratóstenes es un algoritmo que permite hallar todos los números primos menores que un número natural dado. Este enlace nos mostrará la ejecución de este algoritmo sobre el número que en la URL figure como argumento, en este caso 100.

Fibonacci

En matemáticas, la sucesión o serie de Fibonacci es la siguiente sucesión infinita de números naturales. La sucesión comienza con los números 0 y 1.​ A partir de estos, «cada término es la suma de los dos anteriores», es la relación de recurrencia que la define. En este caso este link nos muestra el término n-ésimo de la sucesión siendo n el número que pasemos como argumento en la URL, en este caso 20

Expresiones Regulares

Esta pequeña aplicación trata de identificar, con el uso de expresiones regulares:

  • Cualquier palabra seguida de un espacio y una única letra mayúscula.
  • Correos electrónicos válidos.
  • Números de tarjeta de crédito cuyos dígitos estén separados por - o espacios en blanco cada paquete de cuatro dígitos.
